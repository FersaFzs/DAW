Introducción a las páginas web y el codigo HTML



1. ¿Qué es una Página Web?

Una Página Web es un documento que puede contener texto, imágenes, hipervínculos y elementos multimedia.

Las páginas Web son almacenadas en unas máquinas llamadas servidores. Los usuarios podemos visitarlas desde un ordenador con conexión a Internet utilizando un programa cliente denominado Navegador.

Esta comunicación entre los dos ordenadores se realiza gracias al protocolo de comunicaciones “http” de ahí que las direcciones que escribimos en la Barra de direcciones de un navegador comiencen siempre por http://

Esto indica al navegador que lo que deseamos visualizar es una página Web, ya que la visualización de páginas Web no es el único cometido de un navegador.

El elemento que nos permite navegar por Internet, es el Enlace o hipervínculo.

Los enlaces o hipervínculos, son los que convierten a un documento en Hipertexto. El hipertexto no es más que un documento que contiene enlaces.

Al activar un enlace, podemos visitar una página Web almacenada en cualquier lugar del mundo, o simplemente saltar a otra parte de la misma página que estábamos visualizando.




2. ¿Qué es HTML?

HTML o Lenguaje de marcas de hipertexto es el lenguaje que se utiliza en la construcción de páginas Web. HTML no es un lenguaje de programación, sino un lenguaje de creación de hipertexto.

Como ya hemos visto, un hipertexto no es más que un documento navegable, es decir un documento que contiene hipervínculos.

HTML permite incluir texto, imágenes, vídeos, sonidos e hipervínculos en un página Web. Este lenguaje se basa en un sistema de Etiquetas o Tags que confieren formato a los elementos incluidos en una página Web.

Los documentos confeccionados con este lenguaje, pueden ser visualizados en cualquier ordenador, independientemente del sistema operativo que utilicen. Para visualizar estos documentos, únicamente debemos tener instalado un browser o navegador.

Los navegadores más utilizados son Microsoft Internet Explorer, que se utiliza unicamente bajo Windows, y Netscape Navigator, que puede ser utilizado bajo cualquier sistema operativo.




3. ¿Por qué aprender HTML?

Existen herramientas de edición de páginas Web que facilitan enormemente esta tarea. Las más conocidas son Microsoft Frontpage y Macromedia Dreamweaver. Estas herramientas tienen un interfaz de usuario muy sencillo de utilizar y que permiten crear páginas Web sin necesidad de tener conocimientos de HTML. El motivo por el que debemos aprender este lenguaje es que, en ocasiones, se pueden presentar situaciones que no seamos capaces de resolver mediante estas herramientas, pero sí modificando el código que éstas generan. También debemos conocer HTML, si deseamos crear páginas Web dinámicas incluyendo en éstas fragmentos de código en otros lenguajes como JavaScript o ASP.



4. Los colores

Los colores en HTML se expresan mediante el sistema RRGGBB. En este sistema debemos expresar la cantidad de cada uno de los tres colores primarios: Rojo, Verde y Azul que posee el color que queremos definir. Las letras provienen del nombre de los colores en inglés, la R para Red, la G para Green y la B, para Blue.

La cantidad de color se expresa en números hexadecimales de dos cifras, luego la menor cantidad de color estará expresada por “00” y la mayor por “FF”, puesto que en base hexadecimal, los dígitos van del cero al 9 y de la A a la F.

La sintaxis de este atributo es la de la imagen, donde RR es la cantidad de Rojo que posee el color definido, GG la cantidad de Verde y BB la cantidad de Azul .

Siguiendo este criterio el color Rojo estaría expresado por el número hexadecimal FF0000, el color Verde por 00FF00 y el Azul por 0000FF. El color negro se define como la ausencia de color, es decir 000000, y el color blanco como la presencia total de color FFFFFF.

El resto de colores se expresan como una mezcla de éstos, aumentando o disminuyendo la cantidad de intensidad de cada uno de los colores. Por ejemplo el color Amarillo estaría definido como FFFF00.

La mayoría de los navegadores son capaces de interpretar algunos colores por su nombre en inglés, como por ejemplo Red para el color Rojo, Blue,para el Azul o Green, para el Verde .

Cuando definamos un color utilizando su nombre inglés, debemos escribirlo sin comillas ni almohadillas.




5. Recomendaciones antes de empezar

Antes de empezar a crear nuestro sitio Web, es recomendable generar una estructura de carpetas para mantener organizado el material que emplearemos en su construcción.Esto facilita enormemente el trabajo a la hora de modificar las páginas del sitio Web o reutilizar el material empleado.

También debemos crear dentro de Public HTML una carpeta para las imágenes, imágenes de fondo, barras, viñetas y demás elementos que emplemos en las páginas. De este modo tendremos todo el material organizado y estructurado.
